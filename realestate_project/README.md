# RealEstate Flask Project

تشغيل (خطوات سريعة):

1. إنشاء بيئة افتراضية:
   python -m venv venv
   source venv/bin/activate   # على mac/linux
   venv\Scripts\activate      # على ويندوز

2. تثبيت الحزم:
   pip install -r requirements.txt

3. تشغيل التطبيق:
   python app.py

ثم افتح: http://127.0.0.1:5000/
